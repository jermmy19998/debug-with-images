# debug-with-images
a debug tool for computer 
This tool will save image while debugging
Support type :
- numpy
- dicom
- torch tensor

## Usage
```
from debug_with_img import dwi
dwi(img,'img0':str,save_path)
```

save_path default to './img_debug'
